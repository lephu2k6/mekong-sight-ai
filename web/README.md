# Web Dashboard

Frontend dashboard for Mekong Sight AI, built with React, TypeScript, and Vite.

## Purpose

The web application is the main operations interface for:

- dashboard monitoring
- farm management
- IoT device visibility
- alerts and notifications
- AI analysis and decision support
- user administration

## Development

```bash
npm run dev --workspace=web
```

## Build

```bash
npm run build --workspace=web
```

## Production

The production web container is defined in [web/Dockerfile](./Dockerfile) and served with Nginx.

For the full project overview and deployment flow, see the root [README](../README.md).
